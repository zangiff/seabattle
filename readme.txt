МОРСКОЙ БОЙ
Игра "Морской бой" собственной разработки в одиночном режиме, написанная по полностью оригинальной концепции с широким использованием возможностей нового Javascript (ES6 и выше).
Размер поля - 20 на 20 клеток. Количество кораблей - 16 (можно легко добавить новые корабли / удалить существующие).
Стрельба возможна как через ввод координат в специальное поле, так и с помощью кликов ЛКМ по нужной ячейке.
Флот противника состоит из 1 линкора (4 палубы), 3 крейсеров (3 палубы каждый), 5 эсминцев (2 палубы) и 7 субмарин (одноклеточные существа).
В игре присутствуют темная и светлая темы (на выбор играющего), привлекательный визуальный интерфейс, информационные панели. Масса возможностей для ведения боя, в т.ч с использованием подсказок. Сохранен вывод расположения кораблей в консоль.
Используется богатая цветовая гамма для подстветки полей в процессе ведения боя в зависимости от результатов стрельбы.
Все иконки используются в виде SVG со вставкой в HTML-код и дальнейшей настройкой внешнего вида.
В процессе создания игры какие-либо библиотеки не использовались, игра полностью написана на нативном Javascript. Шрифты находятся в отдельной папке. Для полноценной работы приложения достаточно оффлайн-режима: загрузка каких-либо сторонних файлов из Сети не требуется.
Веб-интерфейс полностью адаптивен и рассчитан под любые экраны от 320 до 1920 пикселей шириной.
